
# InACTually NodeCreator for InACTually Engine
a MediaServer for driving all technology in\
interactive theater for actual acts

This is the InACTually NodeCreator for [InACTually Engine](https://github.com/InACTually/InACTually-Engine) (in another repo) to simplify the creation of new nodes while programming :)

<br>

## 🛠️ Setup – How to compile

To make it compile... pray.
At the moment, only a project file for VisualStudio2022 is available.
A CMake workflow would be a dream.

You need to clone the [InACTually Engine](https://github.com/InACTually/InACTually-Engine) rercusively `--recursive`, as the NodeCreator is a submodule of the Engine.
The NodeCreator does need some files from the Engine, so it does not build/run sololy.
